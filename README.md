# unicorn-jobs

Code repository for the unicorn-jobs project.

## Pre-requisites

Node version 18 or higher

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
